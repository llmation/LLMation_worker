# RAG系统测试文档

本文档专门用于测试检索增强生成(RAG)系统的功能。

## 独特知识点

1. LLMation是一个基于大语言模型的自动化工作流平台。
2. LLMation_worker是LLMation的后端组件，负责处理API请求并与大语言模型通信。
3. 该项目使用了以下关键技术：
   - Flask作为Web框架
   - LangChain用于构建AI应用
   - DashScope用于生成文本嵌入
   - Gemini作为大语言模型

## 独特例子

在LLMation_worker项目中，有一个特殊的测试用例：
- 测试ID: TEST-RAG-2023
- 测试内容: "独角兽公司的股票代码是UNICORN-123"
- 测试结果: "成功检索并回答关于独角兽公司的问题"
